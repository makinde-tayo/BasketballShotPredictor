# Basketball Shot Predictor

Simple python program to keep track of shots and predict mid air the chance of a shot making or missing 

![](https://miro.medium.com/v2/resize:fit:1400/1*tSpvo1iHGaeRDsDV8-mWZg.png)



## Methods
This program tracks the ball using color detection, detects the hoop with a YOLO model, predicts the shot path using a polynomial regression model
